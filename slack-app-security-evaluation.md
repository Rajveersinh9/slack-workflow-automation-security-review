# Slack App Security Evaluation Template

---

## App Name:
Sample Analytics Bot

---

## Permissions Requested

- Read public channel messages
- Post messages
- Access user profile information

---

## Risk Assessment

Risk Level: Medium

Concerns:
- Broad message reading scope
- Access to user metadata
- Potential data storage outside organization

---

## Security Review Questions

- Does the app follow least privilege?
- Where is data stored?
- Is there an audit trail?
- Can permissions be restricted to specific channels?

---

## Recommendation

Approve with restricted access.
Limit to designated channels.
Require quarterly review.
